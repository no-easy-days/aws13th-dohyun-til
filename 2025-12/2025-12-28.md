# PYTHON
## about python

* **인터프리터 언어**: 한 줄씩 실행됨 (컴파일 X)
* **동적 타이핑**: 변수 선언 시 타입 지정 안 함
* **들여쓰기(Indentation)가 문법**임 → 매우 중요

```python
if True:
    print("참")
# 들여쓰기 틀리면 에러
```

# 2. 변수와 자료형

## 2.1 변수 선언


* 값이 바뀌면 타입도 바뀔 수 있음

| 자료형   | 설명  | 예시              |

# 4. 제어문

## 4.1 if 문

```python
if x > 0:
    print("양수")
elif x == 0:
    print("0")
else:
    print("음수")
```

* **elif** = else if
* 괄호 없음, **콜론 + 들여쓰기 필수**

---

## 4.2 for 문

```python
for i in range(5):
    print(i)
```

* `range(5)` → `0,1,2,3,4`

```python
for i in range(1, 10, 2):
    print(i)  # 1,3,5,7,9
```

---

## 4.3 while 문

```python
i = 0
while i < 5:
    print(i)
    i += 1
```

---

## 4.4 break / continue

```python
for i in range(10):
    if i == 5:
        break   # 반복 종료
```

```python
for i in range(10):
    if i % 2 == 0:
        continue  # 아래 코드 스킵
    print(i)
```

# 8. 클래스(Class) & 객체지향

## 8.1 클래스 정의

```python
class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age
```

* `__init__` = 생성자
* `self` = 객체 자신

---

## 8.2 객체 생성

```python
p1 = Person("Tom", 20)
print(p1.name)
```

---

## 8.3 메서드 정의

```python
class Person:
    def greet(self):
        print(f"Hello, I'm {self.name}")
```

```python
p1.greet()
```

---

## 8.4 클래스 변수 vs 인스턴스 변수

```python
class Student:
    school = "ABC University"  # 클래스 변수

    def __init__(self, name):
        self.name = name       # 인스턴스 변수
```

---

## 8.5 상속

```python
class Animal:
    def speak(self):
        print("sound")

class Dog(Animal):
    def speak(self):
        print("bark")
```
